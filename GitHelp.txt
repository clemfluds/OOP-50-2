Команды на один раз для устройства ПК/Ноутбук
git config --global user.name "Your name"
git config --global user.email "your-email@example.com"

Команды на один раз для проекта
git init
git remote add origin https://github.com/clemfluds/OOP-50-2.git

Команды которые часто пишутся в проекте
git add . | Команда копирует и сохраняет в буфер (Невидимый)
git commit -m "commit message"| Команда сохраняет изменения в файле .
git push origin main|Команда отправляет изменения в репозиторий и сохраняет
git pull origin master | Команда для получения изменений из репозитория
git branch| Команда для просмотра списка веток
git status| Команда для просмотра состояния проекта

git rm -r --cached . idea

Базовые команды терминала
clear | Очистка терминала
cd ../ | Переход на верхний уровень вложенности
сd ~ | Переход в корневой каталог
cd name | Переход в каталог name
ls | Просмотр содержимого каталога
ls -a | Просмотр содержимого каталога с показом скрытых файлов
mkdir name | Создание каталога name
rmdir name | Удаление каталога name
rm name | Удаление файла name
touch name - name.txt | Создание пустого файла name
nano | Просмотр содержимого файла в текущем
pwd | Просмотр текущего каталога
